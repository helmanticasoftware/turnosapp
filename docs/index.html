Politica de privacidad

TurnosApp puede tratar datos necesarios para el funcionamiento de la aplicacion, la autenticacion, la sincronizacion entre dispositivos, la gestion de calendarios y turnos, las suscripciones premium y, en su caso, la publicidad de la version gratuita.

Responsable

App: TurnosApp
Desarrollador: Helmantica Software
Contacto: soporte@helmanticasoftware.com

Datos que podemos tratar

  • Nombre, correo electronico u otros datos de cuenta si el usuario inicia sesion.
  • Calendarios, turnos, notas, configuraciones y otros datos creados dentro de la app.
  • Datos tecnicos minimos para seguridad, mantenimiento y sincronizacion.
  • Datos relacionados con suscripciones y compras si se activan funciones premium.
  • Datos publicitarios cuando corresponda en la version gratuita.

Finalidad del tratamiento

  • Permitir autenticacion y sincronizacion.
  • Guardar y restaurar datos del usuario.
  • Gestionar suscripciones y restauraciones de compra.
  • Mostrar anuncios en la version gratuita, cuando aplique.
  • Proteger la app frente a fraude, abuso o incidencias tecnicas.

Terceros o servicios relacionados

  • Supabase, para autenticacion, base de datos y sincronizacion.
  • Google, si el usuario usa inicio de sesion con Google.
  • Google AdMob, si la app muestra anuncios.
  • RevenueCat y Google Play, para compras y suscripciones.

Eliminacion de cuenta

Si quieres eliminar tu cuenta de TurnosApp, puedes solicitarlo escribiendo al correo de soporte con el asunto Eliminar cuenta TurnosApp.

Como solicitarlo

  • Escribe a soporte@helmanticasoftware.com.
  • Indica el correo con el que accediste a la app.
  • Si es necesario, podremos pedir una verificacion minima de titularidad.

Que se elimina con la cuenta

  • Perfil de usuario asociado a TurnosApp.
  • Configuraciones personales y preferencias.
  • Calendarios, turnos, notas y datos asociados a la cuenta.

Eliminacion de datos

La solicitud de eliminacion de cuenta incluye tambien la eliminacion de los datos personales y de uso asociados a esa cuenta, salvo los registros minimos que deban conservarse temporalmente por motivos legales, tecnicos o de seguridad.

Plazo de gestion

Las solicitudes se gestionan normalmente en un plazo maximo de 30 dias desde su recepcion.

Datos que pueden conservarse temporalmente

  • Registros tecnicos minimos para seguridad y prevencion de fraude.
  • Informacion estrictamente necesaria si existe una obligacion legal de conservacion.

Contacto

Para cualquier solicitud relacionada con privacidad, eliminacion de cuenta o eliminacion de datos, escribe a soporte@helmanticasoftware.com.
